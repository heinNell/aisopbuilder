## Generating New SOPs

### Step-by-Step Guide

1. **Click "Generate SOP" Tab**
   - You'll see a form with several fields

2. **Fill in the Required Information**
   
   **SOP Topic/Title*** (Required)
   - Be specific and descriptive
   - Example: "Customer Complaint Handling"
   - Example: "Equipment Calibration Procedure"
   - Example: "Data Backup and Recovery"

   **Department** (Optional)
   - Specify which department will use this SOP
   - Examples: Customer Service, IT, Manufacturing, HR
   - Helps tailor the SOP to specific organizational needs

   **Complexity Level**
   - **Simple**: Basic, straightforward procedures (5-10 steps)
   - **Medium**: Standard operations with multiple steps (10-20 steps)
   - **Complex**: Advanced procedures with sub-processes (20+ steps)

   **Additional Context** (Optional but recommended)
   - Industry-specific requirements
   - Compliance standards (ISO, OSHA, FDA, etc.)
   - Special considerations or constraints
   - Related processes or dependencies
   - Target audience skill level

3. **Click "Generate SOP"**
   - The AI will process your request (usually 10-30 seconds)
   - You'll see a loading animation

4. **Review the Generated SOP**
   - The complete SOP appears below the form
   - Includes all standard sections:
     - Document Information
     - Purpose and Objective
     - Scope and Applicability
     - Definitions
     - Responsibilities
     - Materials/Equipment
     - Detailed Procedures
     - Safety Considerations
     - Quality Control
     - References
     - Revision History

5. **Save Your Work**
   - **Copy**: Click the "Copy" button to copy to clipboard
   - **Download**: Click "Download" to save as Markdown file

### Example: Generating a Simple SOP

**Topic:** "Email Signature Update"  
**Department:** "IT"  
**Complexity:** "Simple"  
**Additional Context:** "Procedure for employees to update their email signatures in Outlook"

This will generate a concise SOP with step-by-step instructions.

### Example: Generating a Complex SOP

**Topic:** "Incident Response and Management"  
**Department:** "IT Security"  
**Complexity:** "Complex"  
**Additional Context:** "Must comply with ISO 27001. Include detection, containment, eradication, recovery, and post-incident review phases."

This will generate a comprehensive, multi-phase SOP with detailed procedures.

## Improving Existing SOPs

### When to Use This Feature

- You have an existing SOP that needs updating
- Your current SOP lacks structure or clarity
- You want to enhance professionalism
- You need to add missing sections
- You want to improve compliance alignment

### Step-by-Step Guide

1. **Click "Improve SOP" Tab**

2. **Paste Your Existing SOP**
   - Copy and paste the complete text of your current SOP
   - Can be from any format (plain text, Word copy, etc.)
   - Character count displays below the text area

3. **Select Improvement Focus**
   - **Overall Quality**: Comprehensive improvement (default)
   - **Formatting & Structure**: Focus on organization
   - **Language & Professionalism**: Enhance writing style
   - **Detail & Completeness**: Add more thorough explanations
   - **Compliance & Safety**: Emphasize regulatory requirements

4. **Click "Improve SOP"**
   - Processing takes 15-40 seconds depending on length
   - Longer SOPs may take more time

5. **Review the Improved Version**
   - The AI provides:
     - Enhanced version with better structure
     - Improved language and clarity
     - Added missing sections
     - Better formatting
     - Summary of improvements made
     - Recommendations for further enhancement

6. **Compare and Save**
   - Keep the original for reference
   - Review all changes carefully
   - Download the improved version

### Best Practices for Improvement

- Include the complete original SOP for context
- Don't just paste a single paragraph
- Specify any industry-specific requirements in the text
- Review the improved version for accuracy
- Merge improvements with your original if needed

## Analyzing SOPs

### What You Get

The analyzer provides comprehensive metrics:

1. **Quality Scores (0-100)**
   - Completeness Score
   - Clarity Score
   - Structure Score
   - Compliance Score

2. **Qualitative Assessment**
   - 3-5 key strengths
   - 3-5 areas needing improvement
   - List of missing sections
   - Specific actionable recommendations

3. **Executive Summary**
   - High-level overview
   - Key procedural steps
   - Critical points and warnings
   - Expected outcomes

### Step-by-Step Guide

1. **Click "Analyze SOP" Tab**

2. **Paste Your SOP**
   - Complete document recommended
   - Works with partial documents but less accurate

3. **Click "Analyze SOP"**
   - Takes 20-45 seconds
   - Generates both detailed analysis and summary

4. **Review Results**
   - **Detailed Analysis Tab**: Full assessment with scores
   - **Executive Summary Tab**: Concise overview

5. **Use the Insights**
   - Identify weak areas to improve
   - Understand what's missing
   - Follow recommendations
   - Track improvements over time

### Interpreting Scores

- **90-100**: Excellent - Publication ready
- **80-89**: Good - Minor improvements needed
- **70-79**: Adequate - Several areas to improve
- **60-69**: Fair - Significant improvements needed
- **Below 60**: Needs major revision

## Uploading Documents

### Supported File Types

- **PDF** (.pdf) - Most common format
- **Word** (.doc, .docx) - Microsoft Word documents
- **Text** (.txt) - Plain text files
- **Markdown** (.md) - Markdown formatted documents

### Maximum File Size: 10MB

### Step-by-Step Guide

1. **Click "Upload & Process" Tab**

2. **Upload Your File**
   - Click the upload area or drag and drop
   - File name and size will display
   - Supported formats listed at bottom

3. **Click "Upload & Analyze"**
   - File is uploaded and processed
   - Text is extracted automatically
   - Immediate analysis is performed

4. **Review Results**
   - Extracted text length shown
   - Quick analysis provided
   - File automatically available for improvement

5. **Next Steps**
   - Switch to "Improve SOP" tab (content auto-loaded)
   - Content already copied for you to enhance
   - Or copy analysis for your records

### Upload Tips

- Ensure text is selectable in PDFs (not scanned images)
- Clean formatting works best
- Remove headers/footers if problematic
- Test with a small file first

2. **Provide Context**: Mention industry standards, regulations, or requirements

1. **Complete Documents**: Provide the full SOP for best context
2. **Clear Focus**: Select the most relevant improvement focus
3. **Review Carefully**: AI improvements should be verified for accuracy
4. **Iterate**: You can improve the improved version for refinement

### For Analysis

1. **Full Text**: Analyze complete documents for accurate scores
2. **Track Progress**: Analyze before and after improvements
3. **Use Recommendations**: Follow specific suggestions provided
4. **Regular Reviews**: Periodically analyze your SOPs for quality

## Exporting Your Work

### Copy to Clipboard

1. Click the "Copy" button
2. Paste into your document editor
3. Format as needed in your system

### Download as File

1. Click the "Download" button
2. File downloads as Markdown (.md)
3. Open in any text editor or Markdown viewer
4. Convert to other formats as needed

### Converting Markdown

**To Word:**
- Use Pandoc: `pandoc input.md -o output.docx`
- Or copy/paste into Word and format

**To PDF:**
- Use online converters
- Or print to PDF from Word/browser
- Or use Pandoc: `pandoc input.md -o output.pdf`

**To HTML:**
- Use Markdown editors with HTML export
- Or Pandoc: `pandoc input.md -o output.html`

## Troubleshooting

### "Error generating SOP"

**Causes:**
- OpenAI API key not configured
- Network connectivity issue
- API rate limit reached

**Solutions:**
- Check backend console for errors
- Verify .env file has valid API key
- Wait and try again if rate limited

### Generated SOP is too generic

**Solutions:**
- Add more specific context
- Mention industry standards
- Describe target audience
- Include special requirements
- Try again with more details

### Uploaded file not processing

**Causes:**
- Unsupported file type
- File too large (>10MB)
- Corrupted file
- Scanned PDF (no selectable text)

**Solutions:**
- Check file format is supported
- Reduce file size
- Try converting to text first
- Use OCR for scanned documents

### Improvement seems off-topic

**Solutions:**
- Provide complete original text
- Select more specific focus area
- Add context in the original text
- Review and manually adjust
